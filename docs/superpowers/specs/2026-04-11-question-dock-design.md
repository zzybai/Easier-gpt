# SlimGPT 右上角问题栏设计

## 目标
在 ChatGPT 会话页右上角增加一个问题栏，只展示用户问题摘要，支持点击跳转到对应 turn，并提供展开/折叠切换。

## 范围
- 只收集用户消息，不展示 assistant 回复。
- 以 `turnIndex` 去重，同一轮只保留一条问题摘要。
- 问题栏默认展开，支持用图标按钮折叠与展开。
- 当前视口所在问题高亮。
- 窄屏默认折叠，避免遮挡正文。

## 架构
- 数据层复用 `STATE.messages` 和现有 `getItemText()` / `jumpToTurn()`。
- 新增纯函数模块 `question-dock.js`，负责把消息模型转换为问题列表，便于 Node 测试。
- `content.js` 负责创建右上角浮层、处理折叠状态、高亮和点击跳转。
- `styles.css` 负责浮层卡片、列表、激活态和窄屏折叠样式。

## 交互
- 展开态显示标题、划线眼图标按钮和问题列表。
- 折叠态保留一个小胶囊按钮，按钮内使用划线眼图标。
- 点击问题项后跳到对应 turn，问题栏保持当前展开/折叠状态。

## 验证
- `node --test` 覆盖问题列表构建逻辑。
- 浏览器手工验证展开/折叠、点击跳转、当前项高亮和窄屏默认折叠。
